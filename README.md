# RegistroAsistencias
Backend de la aplicación de Registro de Asistencias, que permitirá a los docentes tomar la asistencia de los estudiantes a cada uno de los cursos que tiene activo cada semestre.

## Tecnologías

- PHP Laravel 5.0
- HTML5, CSS3 y JavaScript
- MySQL
